# EV-Adoption-Analysis
This project analyzes the growth of electric vehicle (EV) adoption and charging infrastructure across major cities

Electric Vehicle Adoption & Infrastructure Analysis

#Project Overview

This project analyzes the growth of electric vehicle (EV) adoption and charging infrastructure across major cities in India from 2020 to 2025. The analysis was performed using Excel for data preparation and Power BI for visualization to identify trends, regional adoption patterns, and infrastructure gaps.


#Objectives

Analyze EV adoption trends over time

Identify cities with the highest EV registrations

Compare charging infrastructure across regions

Detect areas where EV growth exceeds charging station availability


#Tools Used

Microsoft Excel – Data cleaning and preparation

Power BI – Data visualization and dashboard creation

GitHub – Project documentation and portfolio
